# 同棲用家計簿アプリ — design.md

---

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| アプリ名 | ふたりの家計簿 |
| コンセプト | 同棲中の2人が毎日ストレスなく使える、シンプルで楽しい家計簿 |
| ターゲット | 同棲カップル（自分 / パートナー） |
| 主な利用端末 | スマートフォン（ブラウザ） |
| データ保存 | LocalStorage（サーバー不要） |
| 認証 | なし（2人でブックマークして共有） |

---

## 2. 要件定義

### 2-1. 機能要件

#### 支出入力
- 日付・カテゴリ・金額・支払者（自分 / パートナー）・メモ の5項目を入力
- カテゴリは以下の4つ（固定）
  - 食費 / 日用品 / 家賃・光熱費 / 外食・デート
- 入力後すぐ一覧に反映される

#### 支出一覧
- 登録した支出を新しい順に一覧表示
- カテゴリ・支払者でフィルタリング可能
- 1件ずつ削除できる

#### 月次集計
- 今月の合計支出を数字で表示
- 自分 / パートナー それぞれの支払い合計を表示
- カテゴリ別の内訳を **円グラフ**（割合）と **棒グラフ**（金額）で表示

#### 精算機能
- 2人の支払い差額を自動計算
- 「自分がパートナーに ○○円 受け取る / 渡す」と一言で表示
- 精算ボタンを押すと履歴にメモとして残る

#### 月次アーカイブ
- 月が変わると前月データをアーカイブとして保存
- 過去月の集計・一覧を閲覧できる（編集不可）

### 2-2. 非機能要件

- **パフォーマンス**: ページ読み込み1秒以内（静的HTMLのみ）
- **オフライン対応**: インターネット不要で動作（LocalStorage）
- **レスポンシブ**: スマホ幅（375px〜）で最適表示
- **ファイル構成**: `index.html` 1ファイルで完結（CDN利用可）

## 3. 画面構成（ページ設計）

```
index.html
├── ① ホーム画面（今月のサマリー）
│   ├── 今月の合計・2人の内訳
│   ├── カテゴリ別グラフ（円グラフ）
│   └── 精算ボタン
├── ② 入力画面（支出を追加）
│   └── 入力フォーム（日付・カテゴリ・金額・支払者・メモ）
├── ③ 一覧画面（支出リスト）
│   ├── フィルター（カテゴリ・支払者）
│   └── 支出リスト（削除ボタン付き）
└── ④ アーカイブ画面（過去月）
    └── 月選択 → 過去の集計・一覧を表示
```

ナビゲーション: 画面下部のタブバー（ホーム / 追加 / 一覧 / アーカイブ）

## 5. デザイン仕様

### 5-1. デザインコンセプト

> **「毎日ちょっと嬉しくなる、生活に溶け込む家計簿」**

白ベースのクリーンな画面に、やわらかいアクセントカラーを添える。
数字を入力する作業が、ちょっとした楽しみになるような小さなアニメーションを散りばめる。

### 5-2. カラーパレット

| 役割 | カラー | 用途 |
|------|--------|------|
| Primary | `#5B8DEF` | ボタン・アクティブ状態 |
| Accent | `#FF8FAB` | 精算ハイライト・バッジ |
| Background | `#F8F9FC` | ページ背景 |
| Surface | `#FFFFFF` | カード背景 |
| Text Primary | `#1A1A2E` | 見出し・金額 |
| Text Secondary | `#6B7280` | メモ・日付 |
| Border | `#E5E7EB` | カード枠線 |

カテゴリカラー:
- 食費: `#7BC47F`（グリーン）
- 日用品: `#64B5F6`（ブルー）
- 家賃・光熱費: `#FFB74D`（オレンジ）
- 外食・デート: `#F48FB1`（ピンク）

### 5-3. タイポグラフィ

| 用途 | フォント | サイズ |
|------|----------|--------|
| アプリ名・大見出し | Noto Sans JP Bold | 22px |
| 金額表示 | tabular-nums / Bold | 28px |
| 本文・ラベル | Noto Sans JP Regular | 14px |
| 補足・メモ | Noto Sans JP Regular | 12px |

### 5-4. コンポーネント仕様

#### 支出カード
```
┌─────────────────────────────────┐
│ 🛒 食費          2025/06/15     │
│ スーパーで買い物                │
│                    自分  ¥3,200 │
└─────────────────────────────────┘
```
- 左にカテゴリアイコン＋カラーバー
- 右下に支払者バッジ＋金額（大きめ）
- スワイプ or 長押しで削除

#### 精算バナー
```
┌─────────────────────────────────┐
│  💸 今月の精算                  │
│  パートナーに ¥4,500 渡す       │
│              [ 精算完了にする ] │
└─────────────────────────────────┘
```

#### 下部タブバー
- アイコン＋テキスト
- アクティブタブは Primary カラーで強調

### 5-5. アニメーション

| 場面 | アニメーション |
|------|----------------|
| 支出追加 | カードが下からフェードイン（0.3s ease-out） |
| タブ切り替え | コンテンツがフェード（0.2s） |
| 精算ボタン押下 | 軽いバウンス＋カラーフラッシュ |
| 金額入力確定 | 合計数値がカウントアップ |

## 6. 技術スタック

| 分類 | 採用技術 | 理由 |
|------|----------|------|
| 構成 | HTML / CSS / Vanilla JS | サーバー不要・学習コスト低 |
| グラフ | Chart.js（CDN） | 円グラフ・棒グラフが簡単に実装可能 |
| アイコン | Font Awesome（CDN） | カテゴリアイコンに使用 |
| フォント | Google Fonts（Noto Sans JP） | 日本語対応・読みやすい |
| ストレージ | LocalStorage | オフライン・サーバー不要 |
